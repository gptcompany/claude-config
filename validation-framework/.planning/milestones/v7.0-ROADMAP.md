## Milestone Summary

**Key Decisions:**
- Multi-model via OpenRouter for cross-review (avoid vendor lock-in)
- Plan C for autonomous loop (spec-first + auto-validate, simpler than Plan A/B)
- Budget enforcer as systemd timer + textfile collector (lightweight, no extra infra)
- Gemini Flash for cross-review (free tier, good enough for code review)
