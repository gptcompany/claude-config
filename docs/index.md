# Claude Config

Central configuration repository for Claude Code workflows, GSD/Speckit frameworks, and GitHub integrations.

## Key Features

- **GitHub Sync**: Automatic bidirectional sync between GSD/Speckit and GitHub Issues
- **Reusable Workflows**: Shared CI/CD workflows for all organization repos
- **Validation Scripts**: Code quality and planning validation tools
